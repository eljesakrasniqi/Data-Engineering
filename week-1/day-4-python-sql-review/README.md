# Day 4 - Python and SQL Logic Review

## Goal of the Practice
The goal of this practice was to review Python and SQL logic by working with order data.  
In Python, I practiced processing data using lists, dictionaries, filtering, sorting, calculations, and summary operations.  
In SQL, I practiced querying data, filtering records, calculating total amounts, and analyzing business information.

## Files Included
  - `python_orders.py` - Python script that processes order data and performs different tasks:
  - Printing basic order information
  - Filtering orders by status and category
  - Calculating total amounts
  - Sorting orders
  - Creating simple business summaries
