## 1. What dataset did we analyze?

We analyzed an orders dataset (`orders.csv`) containing information about customer orders.

The dataset included:
- Order ID
- Customer name
- City
- Product
- Category
- Quantity
- Price
- Order status
- Order date

The goal was to analyze business performance and extract useful insights from the order data.

---

## 2. What did we do in Python?

In Python, we processed the CSV file and performed data analysis tasks.

We:
- Read the orders data from the CSV file.
- Counted the total number of orders.
- Analyzed order statuses.
- Counted orders by city.
- Calculated total amounts using quantity × price.
- Found important order and revenue information.
- Generated reports from the processed data.

## 3. What did we do in SQL?

In SQL, we created and analyzed the database using two files:

- `setup.sql` → Created tables and inserted the order data.
- `sql_analysis.sql` → Used queries to analyze the data.

We practiced:
- SELECT queries
- Filtering data with WHERE
- Aggregations using COUNT and SUM
- GROUP BY analysis
- Sorting results
- Business-focused queries

SQL practice was completed using SQLiteOnline.com.

Our most important insight was understanding how order data can show business performance.

By analyzing completed orders, revenue, product categories, and customer locations, we can identify which parts of the business perform better and where improvements can be made.
